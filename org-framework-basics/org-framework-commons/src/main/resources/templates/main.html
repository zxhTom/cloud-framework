基本信息

   项目介绍

  • 基于Spring实现类管理；基于Mybatis作为操作sql的桥梁；基于security完成权限管理
  • 设计之初，考虑要领就是仅完善不限制下游操作
  • 同级功能在设计上需要做到完全零感知取代；maltcloud专注于架构底层技术选型的方向。在maltcloud孵化池中两种相同技术可以完全切换而不影响maltcloud内部已有功能

规划

  • 1、专注于技术层面
  • 2、maltcloud内部尽可能自己实现部分框架已有功能，目的是不被具体框架局限
  • 3、持续更新。无法保证有多优秀，可以保证的是截止2021.3每周都会更新
  • 4、尽可能详细的文档输出，目前文档正在更新。待系统发布稳定版本后第一时间发布文档手册
